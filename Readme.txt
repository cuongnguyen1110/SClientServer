
Github: https://github.com/cuongnguyen1110/SClientServer

Prerequiere:
	OS: Linux.
	tool: GCC

How to build:
	Server: go to SClientServer\SocketApp\SocketApp\Server and run 'make'.
			run: 'make clean' to clean source
	Client: go to SClientServer\SocketApp\SocketApp\Client and run 'make'.
			run: 'make clean' to clean source
			
How to run app:
		run MServetApp to run server. enter 'quit' to quit the application.
		run MClientApp to run client. enter 'quit' to quit the application. enter message to send to server.
